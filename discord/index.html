Extra commands

bye Disconnects bot from voice channels.

dc Disconnects bot from voice channels.

die Disconnects bot from voice channels.

join Makes the bot join a voice channel.

music Shows all the music commands.

p Shortcut to play music.

purge Purge (number) deletes a number of messages.

q Shows the music queue.

alias Manage command aliases.

bank Base command to manage the bank.

cleanup Purge messages.

customcommands Create custom commands.

filter Base command to add or remove words from the server filter.

filterset Base command to manage filter settings.

raffle Raffle (giveawaway) group command.

setraffle Set Raffle group command.

report Send a report.

reportset Manage Reports.

netspeed Command measures the bots ping and internet speed.

permissions Command permission management tools.

trivia Start trivia session on the specified category.

triviaset Manage Trivia settings.

Admin commands

addrole Add a role to a user.

announce Announce a message to all servers the bot is in.

announceset Change how announcements are sent in this guild.

editrole Edit role settings.

removerole Remove a role from a user.

selfrole Add or remove a selfrole from yourself.

selfroleset Manage selfroles.

serverlock Lock a bot to its current servers only.
